# 와이어바알리 사전과제
-------------
### 개발환경
- IDE: 이클립스
- OS: Windows 10
- 버전관리 : GIT

###  server
- Spring Boot 2.4.0
- JUnit 5
- Gradle
- Lombok
- thymeleaf
-------------
### Build and Run App
`git clone https://github.com/hongdaepyo/wirebarley.git`

`cd wirebarley/`

`./gradlew clean build`

`java -jar build/libs/wirebarley-0.0.1-SNAPSHOT.jar`
